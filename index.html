Overview

So what is D3? D3 (or more specifically d3.js) is an advanced JavaScript library for creating data visualizations for the web. D3 stands for “Data-Driven Documents.” D3 makes it easy to bind data to JavaScript objects and then use those objects to draw complex svg data diagrams. It also allows you to add transitions and animations to your visualizations. More info about the framework can be found at the official site d3js.org.

A quick note: D3 only supports modern browsers (sorry Internet Explorer!). For the best experience, we recommend you view this portfolio site using the Chrome, Firefox, or Safari browsers.

In starting our analysis of the D3 library we will compare it jQuery, another JavaScript library. We will also look at how transitions and animations in D3 compare to those than can be done with CSS3. This will be done with a simple bar chart, which is probably the most basic visualization one can make in D3.

Next, we will look more at how D3 interacts with various data. We will see how one can add data from outside files, how to sort data, and how to update the data that is bound to the visualizations. For this example, we will create a slightly more advanced stacked bar chart. To explore more complex topics, more complex charts are needed. Using a choropleth map (a map in which regions are colored differently based on data) we will see how to merge multiple data sources into one data object to use with D3. We will also look at the geoJSON data format (and JSON in general), and use that geoJSON to draw complex SVG paths. Another great feature of D3 is the ability to use different layouts to visualize data. We will explore two of them in-depth: pie layout and tree layout. Using pie layout we will create a ring chart (also known as a donut chart) using arc paths. Tooltips will be used to show how interaction can be added to visualizations.
